This project is a modern React image search application built using TanStack Query for efficient server state management and Axios for API requests. It allows users to search for images dynamically, with optimized performance through caching, background refetching, and reduced API calls.

Key Features:
- Dynamic image search based on user input
- Server state management with TanStack Query
- API integration using Axios
- Automatic caching and refetching
- Clean and responsive UI
